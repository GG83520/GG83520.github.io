🔒 Sécurité — Code PIN d'accès
Le code PIN est demandé à l'ouverture de saisie.html et fiche.html. Il est stocké dans l'onglet REF_CONFIG de Google Sheets. Valide pour toute la session navigateur (pas de re-saisie tant que l'onglet reste ouvert).

ℹ️ Comment ça fonctionne
  • Le PIN est stocké dans l'onglet REF_CONFIG de la Google Sheet (clé = PIN).
  • Un PIN correct donne accès pour toute la session navigateur (rechargement = OK, fermeture onglet = re-saisie).
  • Changer le PIN ici invalide les sessions en cours sur les autres pages.
  • Pour désactiver le PIN : supprimer la ligne PIN dans REF_CONFIG directement dans Sheets.
  • Longueur acceptée : 4 à 8 chiffres uniquement.
